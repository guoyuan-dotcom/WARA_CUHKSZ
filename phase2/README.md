# Phase 2 Runtime in WARA

This directory owns Phase 2: modeling, solution-route design, experiment implementation, validation, and experiment evidence.

Phase 2 numbering:
- `2.1` Construct the System Model and Optimization Problem
- `2.2` Check Convexity and Choose a Reformulation Route
- `2.3` Specify the Algorithm, Baselines, and Experiment Plan
- `2.4` Implement and Run the Experiment Code
- `2.5` Verify Results and Promote Paper Figures

Phase 3 is no longer stored as Phase 2 prompt content. Paper writing, references, review, and revision live under the top-level `phase3/` directory.

Compatibility note:
- Active Phase 2 run directories may contain compatibility artifact folders such as `phase3-1` through `phase3-5`.
- Public Phase 3 mirrors and final packages are exposed under `outputs/paper_runs/phase3/<run_id>/` and `outputs/paper_runs/final_papers/<run_id>/`.
