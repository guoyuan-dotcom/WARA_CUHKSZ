# Phase 3 Artifact Boundary

Phase 3 starts after Phase 2 evidence is available. It reads:
- frozen mathematical and algorithm contracts from Phase 2;
- validated experiment evidence from Phase 2.4 and Phase 2.5;
- reference and topic context from Phase 1 handoff artifacts.

Phase 3 writes:
- technical paper sections;
- numerical-results prose;
- abstract, introduction, conclusion, and references;
- review reports, routing decisions, and final revisions.

Phase 3 must not silently alter Phase 2 contracts. If writing or review finds a model, algorithm, or experiment defect, the controller should route the issue back to the owning Phase 2 agent instead of letting Phase 3 invent new mathematics or evidence.
