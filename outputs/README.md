# Runtime Outputs

Generated WARA runs are written here and ignored by git.

- `paper_runs/phase1/<run_id>/`: Phase 1 research-direction artifacts.
- `paper_runs/phase2/<run_id>/`: Phase 2 modeling, algorithm, experiment, and result artifacts.
- `paper_runs/phase3/<run_id>/`: Phase 3 writing, review, repair, and final manuscript artifacts.
- `paper_runs/final_papers/<run_id>/`: final paper packages copied from completed runs.

This clean release intentionally contains only empty output folders. Do not commit generated run folders or local logs.
